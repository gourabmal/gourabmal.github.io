Core Features

Everything in one
intelligent platform

Four major components working together to deliver a seamless hybrid support experience.

🤖

AI-Powered Chatbot

Google Gemini processes every customer query against your product catalog and business data — delivering instant, context-aware responses.

⚡

Supabase Realtime

Messages appear instantly across all clients with no page refresh needed. Built on WebSockets for a true live-chat experience.

👤

Human Agent Takeover

When escalation is needed, agents receive the full conversation on Telegram and reply directly — routed back to the customer in real time.

🔄

n8n Workflow Engine

All orchestration — AI routing, Telegram relaying, session management, and AI suppression — is handled visually through n8n automation.

🛡️

Smart AI Suppression

Once a human agent takes over, AI responses are automatically paused — ensuring a coherent, human-led conversation until resolved.

💬

Embeddable Chat Widget

A lightweight, drop-in chat widget that works on any website. Customers get instant support without leaving the page or creating accounts.

How It Works

From message
to resolution

A complete flow — from the first customer message to a resolved conversation.

01

Customer Opens the Chat Widget

A visitor lands on the website and opens the embedded chat widget. They can ask about products, pricing, features, or services — in plain language.

JavaScript Widget
02

n8n Receives & Orchestrates

Every message hits an n8n webhook endpoint. The workflow decides whether to route to AI processing, session management, or escalation — all without custom code.

n8n Automation
03

Gemini Generates a Contextual Response

Google Gemini AI analyzes the query against configured product catalog and business information — generating a relevant, accurate reply instantly.

Google Gemini AI
04

Supabase Broadcasts in Real Time

The response is written to the database and broadcast via Supabase Realtime WebSockets — appearing in the chat widget immediately, no refresh required.

Supabase Realtime
05

Human Agent Takeover via Telegram

If the customer requests a human — or AI cannot handle the query — the conversation is escalated to a Telegram support group. Agents receive customer message, session ID, and full context.

Telegram Bot API
06

Agent Replies, AI Pauses

The agent replies directly from Telegram. The response is routed back to the correct customer session in real time. AI responses are paused to ensure a clean, human-led conversation.
