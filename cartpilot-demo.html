Core Features

Everything in one
intelligent platform

Four major components working together to deliver a seamless hybrid support experience.

🤖

AI-Powered Chatbot

Google Gemini processes every customer query against your product catalog and business data — delivering instant, context-aware responses.

⚡

Supabase Realtime

Messages appear instantly across all clients with no page refresh needed. Built on WebSockets for a true live-chat experience.

👤

Human Agent Takeover

When escalation is needed, agents receive the full conversation on Telegram and reply directly — routed back to the customer in real time.

🔄

n8n Workflow Engine

All orchestration — AI routing, Telegram relaying, session management, and AI suppression — handled visually through n8n automation.

🛡️

Smart AI Suppression

Once a human agent takes over, AI responses are automatically paused — ensuring a coherent, human-led conversation until resolved.

💬

Embeddable Chat Widget

A lightweight drop-in widget that works on any website. Customers get instant support without leaving the page or creating accounts.

How It Works

From message
to resolution

A complete flow — from the first customer message to a resolved conversation.

01

Customer Opens the Chat Widget

A visitor opens the embedded chat widget and asks about products, pricing, features, or services — in plain language.

JavaScript Widget
02

Message Forwarded to n8n Workflow

Every message hits an n8n webhook. The workflow orchestrates all downstream processing — AI routing, session handling, and escalation logic.

n8n Automation
03

Google Gemini Generates a Response

Gemini AI analyzes the query against configured product catalog and business data — generating a relevant, accurate reply instantly.

Google Gemini AI
04

Supabase Broadcasts in Real Time

The response is pushed via Supabase Realtime WebSockets — appearing in the chat widget immediately, no refresh required.

Supabase Realtime
05

Human Agent Takeover via Telegram

Customer requests a human → conversation routes to a Telegram support group. Agent receives the full context and replies directly from Telegram.

Telegram Bot API
06

Agent Replies — AI Pauses

Agent's Telegram reply is routed back live to the website chat. AI responses are paused for a clean human-led session until the agent closes it.
